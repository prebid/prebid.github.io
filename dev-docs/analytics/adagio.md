#### Registration

The Adagio Analytics adapter requires setup and approval from the Adagio team. Please reach out to <contact@adagio.io> for more information.

#### Example Configuration

```js
  pbjs.que.push(function () {
    pbjs.enableAnalytics({
      provider: 'adagio',
    });
  });
```
