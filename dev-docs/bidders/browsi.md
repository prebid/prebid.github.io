# Sample Ad Unit: For Publishers

```javascript
let videoAdUnit = [
    {
        code: 'videoAdUnit',
        mediaTypes: {
            video: {
                playerSize: [[300, 250]],
                context: 'outstream'
            },
        },
        bids: [{
            bidder: 'browsi',
            params: {
                pubId: '117a476f-9791-4a82-80db-4c01c1683db0', // Publisher ID provided by Browsi
                tagId: '1' // Tag ID provided by Browsi
            }
        }]
    }];
```
