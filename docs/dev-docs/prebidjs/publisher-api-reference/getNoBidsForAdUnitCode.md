---
title: pbjs.getNoBidsForAdUnitCode(adUnitCode)
description: getNoBidsForAdUnitCode API
---


Returns bid requests that resulted in a NO_BID for the specified adUnitCode.  See full documentation at [pbjs.getNoBids()](/dev-docs/publisher-api-reference/getNoBids).

**Kind**: static method of `pbjs`

**Returns**: `Object` - NO_BID bidResponse object

**Request Params:**


| Param | Scope | Type | Description |
| --- | --- | --- | --- |
| adUnitCode | Required | `String` | adUnitCode |
