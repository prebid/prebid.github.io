---
title: pbjs.getAllWinningBids()
description: getAllWinningBids API
---


Use this method to get all of the bids that have won their respective auctions and also rendered on the page.  Useful for [troubleshooting your integration](/dev-docs/prebid-troubleshooting-guide).

* `pbjs.getAllWinningBids()`: returns an array of bid objects that have won their respective auctions and also rendered on the page.
